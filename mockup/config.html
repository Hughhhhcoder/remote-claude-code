Hooks

3 hooks

在 Claude 生命周期事件前后执行命令。用于自动化（如 commit 前格式化）或策略执行。

PostToolUse Edit | Write | NotebookEdit 自动格式化
$ pnpm prettier --write "$CLAUDE_FILE_PATHS"
匹配 **/*.{ts,tsx,js,jsx,md} · 上次执行 2 分钟前
PreToolUse Bash 危险命令检查
$ .claude/hooks/check-dangerous-cmd.sh
阻止 rm -rf /、git push --force 到主分支等
Stop on session end 推送完成通知到手机
$ rcc push "Claude 完成任务: $CLAUDE_LAST_MSG"
任务结束后推送到所有配对设备

权限策略

允许/拒绝规则控制 Claude 哪些操作能直接执行、哪些需要审批。远程会话默认更严格。
