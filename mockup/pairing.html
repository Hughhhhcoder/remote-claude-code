① 启动
一条命令
rcc-host start 会拉起 cloudflared 隧道，分配固定子域名，并在浏览器打开配对页。
② 扫码
二维码或 6 位码
码内包含一次性 token + 主机公钥。扫码后客户端用 ECDH 和 Host 协商出 E2E 密钥。
③ Face ID
创建 Passkey
WebAuthn 标准，私钥存在 Secure Enclave。以后每次连接免密，只需 Face ID。
④ 就绪
长期免密
设备登记在 Host 的 trust store 里。丢手机随时从电脑端吊销。
